# Kittinat's Simple Portfolio

A clean and professional portfolio website showcasing my skills, projects, and experiences.

## Features
- **Responsive Design:** Optimized for all screen sizes.
- **Expertise Pillars:** Dedicated sections for ArcGIS, Offshore Operations, and Site Investigation.
- **Modern Dark Theme:** High-contrast technical aesthetic.

## Technologies
- HTML5 / CSS3 (Vanilla)
- Google Fonts (Inter)
- Git / GitHub Pages

## Project Structure
- `Kittinat_portfolio/` - Portfolio source files
  - `index.html` - Main portfolio page
  - `style.css` - Custom dark theme styling
- `README.md` - Project documentation


## 📝 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/KittinatTa/Kittinat-Simple-Portfolio.git
   ```
2. Open `Kittinat_portfolio/index.html` in your browser.
